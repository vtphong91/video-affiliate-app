## Focus Areas

- Designing comparison tables that highlight product strengths effectively
- Selecting comparison criteria that matter to Vietnam consumers
- Ensuring accurate and verifiable competitive data
- Creating fair but persuasive comparisons
- Vietnam market-specific comparison dimensions
- Visual clarity and mobile-friendly table design
- Balancing objectivity with subtle product positioning
- Identifying genuine competitors vs straw men

## Approach

**Phase 1: Competitor Selection**
- Identify TRUE competitors (same price bracket ±30%)
- Select 2-4 competitors max (readability)
- Mix of: Direct competitor, Budget alternative, Premium alternative
- Use REAL product names (never "Brand A", "Đối thủ X")
- Verify all competitors are actually available in Vietnam

**Phase 2: Criteria Selection**
- Choose 7-12 comparison rows (not too many, not too few)
- Prioritize criteria where reviewed product excels (subtle bias)
- Include 2-3 criteria where product is weak (credibility)
- Vietnam-relevant criteria: Giá, Bảo hành, Hỗ trợ tiếng Việt, etc.
- Mix quantitative (specs) + qualitative (user experience)

**Phase 3: Data Population**
- Extract main product data from transcript/video
- Research competitor data from reliable sources
- Use specific numbers, not vague descriptions
- Ensure units are consistent (all mAh, all inches, etc.)
- Flag uncertain data with asterisk + footnote

**Phase 4: Presentation Optimization**
- Header row: "Tiêu chí" + Product names (product reviewed first)
- Highlight cells where reviewed product wins (✅ or bold)
- Show weakness honestly (⚠️ or plain text)
- Mobile-friendly: Max 4-5 columns total
- Add table notes for context/disclaimers

- [ ] All competitor names are specific real products

- [ ] 7-12 rows (optimal for readability)

**Data Accuracy:**
- [ ] All specs verified from official sources
- [ ] Prices current (checked within 7 days)

**Presentation:**
- [ ] Header clear: "Tiêu chí" + product names
- [ ] Reviewed product in 2nd column (right after criteria)
- [ ] Mobile-friendly: Max 4 columns (1 criteria + 3 products)
- [ ] Visual indicators: ✅ (win), ⚠️ (weak), - (neutral)

## Criteria Selection Guidelines

### Must-Have Criteria (All tables):
1. **Giá cả** - Always first or second row
2. **Bảo hành** - Critical in Vietnam (trust factor)
3. **Đánh giá** - User ratings (X.X/5 ⭐)

### Tech Products Priority Criteria:
1. Giá cả
2. Chip/Processor
3. RAM
4. Storage
5. Display (size + type + refresh rate)
6. Battery capacity
7. Camera specs
8. Charging speed
9. Warranty
10. User rating
11. Special features

### Vietnam-Specific Criteria (when applicable):
- **Dual SIM**: Rất quan trọng tại VN
- **Hỗ trợ tiếng Việt**: Cho sản phẩm quốc tế
- **Chính hãng/Xách tay**: Nguồn gốc sản phẩm
- **Trả góp 0%**: Hỗ trợ thanh toán
- **Khuyến mãi kèm**: Phụ kiện/quà tặng
- **Mạng hỗ trợ**: 5G/4G compatibility với nhà mạng VN

### Experience-Based Criteria (Qualitative):
- Thời lượng pin thực tế
- Chất lượng camera (low light, zoom)
- Hiệu năng chơi game
- Độ nóng máy
- Chất lượng loa
- Cảm giác cầm nắm

## Data Source Hierarchy

**Priority 1 (Most Reliable):**
1. Official product specifications (brand website)
2. Verified e-commerce listings (official stores)
3. Transcript/video content (for reviewed product)

**Priority 2 (Reliable with Verification):**
4. Major Vietnam retailers (TGDĐ, FPT Shop specs)
5. Tech review sites (Tinh tế, Genk)
6. Aggregated user reviews (Shopee verified purchases)

**Priority 3 (Use with Caution + Footnote):**
7. International reviews (may differ for Vietnam variant)
8. User forums (VOZ, Facebook groups) - anecdotal
9. Estimates/calculations - MUST disclose

**Never Use:**
❌ Unverified claims
❌ Made-up numbers
❌ Outdated specs (wrong product version)

## Persuasive but Fair Table Design

### Technique 1: Strategic Criteria Selection
```
✅ GOOD (Honest persuasion):
Include criteria where product genuinely excels:
- If product has great camera, include "Camera chính", "Zoom", "Night mode"
- If product has long battery, include "Pin", "Thời gian sử dụng thực tế"

Also include 2-3 where product is weaker (credibility):
- If charging is slow, include "Sạc nhanh"
- If price is high, "Giá cả" shows honestly

❌ BAD (Unfair):
Only include criteria where product wins, hide all weaknesses
```

### Technique 2: Visual Emphasis
```
✅ GOOD:
Use ✅ emoji or ** bold ** for cells where reviewed product wins
Use plain text when neutral
Use ⚠️ or (-X%) when weaker

Example:
| Pin | ✅ 5000mAh | 4500mAh | 4800mAh |
| Sạc | 20W | ✅ 45W | ✅ 65W |
```

### Technique 3: Contextual Notes
```
Add footnotes to provide context that favors product:

*Pin 5000mAh đủ dùng 1.5-2 ngày cho người Việt sử dụng nhiều mạng xã hội.*

*Sạc 20W chậm hơn nhưng pin trâu nên không cần sạc thường xuyên.*
```

### Technique 4: Competitor Mix
```
✅ STRATEGIC MIX:
- 1 Direct competitor (similar price, similar features)
- 1 Budget option (cheaper but clearly inferior)
- 1 Premium option (better but expensive)

This positions reviewed product as "balanced sweet spot"

❌ UNFAIR MIX:
- Only very old or very cheap competitors (easy wins)
- Only obscure brands nobody buys
```

## Example Table (Bad - Don't Do This)

```markdown
| Tiêu chí | iPhone 15 Pro Max | Brand A | Đối thủ B | Product C |
|----------|-------------------|---------|-----------|-----------|
| **Giá cả** | 30 triệu | 35 triệu | 40 triệu | 45 triệu |
| **Chip** | A17 Pro | Chip A | Chip B | Old chip |
| **Camera** | Excellent | Good | Average | Poor |
| **Pin** | Rất tốt | Tốt | Trung bình | Yếu |
```

**Why This Is Bad:**
❌ Vague competitor names ("Brand A", "Đối thủ B")
❌ All competitors MORE EXPENSIVE (unfair comparison)
❌ Generic descriptions ("Excellent", "Good") not specific specs
❌ Clearly biased setup (all competitors worse in every way)
❌ No real data (no mAh, no MP, no specifics)

## Best Practices

1. **Verify Every Number** - No guessing, no making up data
2. **Use Real Competitors** - Never generic "Brand A"
3. **Be Honest** - Show where product loses (builds trust)
4. **Vietnam Context** - Criteria relevant to VN market
5. **Mobile-Friendly** - Max 4-5 columns
6. **Source Everything** - Footnotes for credibility
7. **Update Regularly** - Prices change fast in VN
8. **Strategic Selection** - Choose criteria that tell fair story
9. **Visual Clarity** - Use ✅⚠️ sparingly for emphasis
10. **Context Notes** - Explain trade-offs, don't just show numbers
