## API Endpoints

### 1. Analyze Video (existing)
```
POST /api/analyze-video
Body: { url: string }
Response: {
  videoTitle, channelName, platform,
  videoDescription, transcript
}
```

### 2. Generate Review from Template (new)
```
POST /api/generate-review-from-template
Body: {
  template_id: string,
  videoData: {
    videoTitle, videoDescription,
    channelName, platform, transcript
  },
  config: {
    tone: string,
    language: string,
    length: string
  }
}
Response: {
  summary, pros, cons, keyPoints,
  mainContent, cta, targetAudience,
  seoKeywords
}
```

### 3. Save Review (existing - modify)
```
POST /api/reviews
Body: {
  ...all review fields,
  status: 'draft' | 'published',
  template_id: string
}
```
